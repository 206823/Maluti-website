# Maluti TVET College Website Revamp (Phase 2 Assessment)

## Project Overview
This project is part of The Battlefield Phase 2 assessment. The goal is to design and prepare a frontend concept and system structure for the Maluti TVET College website revamp.

## Objectives
- Design modern UI/UX using Figma
- Create a basic frontend structure
- Set up a repository
- Prepare API structure (skeleton)
- Configure hosting environment
